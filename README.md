I'm challenged to do authentication with firebase, react, node and mongodb for full stack developer position.
```
$ npm install
```

Then, navigate to the client directory and run the following command:

```
recommended
$ yarn
```
or
```

```

### Running the application

*To run this application, you must have MongoDB up and running*

To run the application, use the following command:

```
$ npm run dev
```
